## 📌 Release Overview & PS5 Console OS Update (V1.4.0)
ZERO 2 Version 1.4.0 transforms your device into a next-generation PS5-inspired gaming console dashboard. Featuring cinematic ambient background glows, animated activity card carousels, console storage visualizers, and universal controller/keyboard DPAD navigation.

---

## 🔥 Key Features & Technical Architecture in V1.4.0

### 1. 🎮 PS5 Console Dashboard (`Ps5ConsoleDashboard.kt`)
- **Dynamic Ambient Background System:** Real-time radial ambient glow engine that adapts to the focused activity card.
- **Animated Activity Card Carousel:** Horizontal card carousel with 350ms `FastOutSlowIn` scaling transitions (Focused: `1.0` scale, Unfocused: `0.88` scale).
- **Glass Top Information Header:** Glassmorphic panel with live digital clock, user profile badge, Wi-Fi status, and mode switcher.

### 2. 💾 Console Storage Screen (`ConsoleStorageScreen.kt`)
- **Storage Breakdown Visualizer:** Console-style storage analyzer showing Games & Apps, Media Gallery, Saved Data, and System storage distribution bars.

### 3. ⚙️ Quick Control Center (`ConsoleControlCenter.kt`)
- **Console Quick Menu:** Spring-animated bottom sheet panel for instant control over Wi-Fi, Bluetooth, Volume, Brightness, Storage, and Power options.

### 4. 🕹️ Universal Gamepad & Keyboard Navigation
- **Landscape Console Mode:** Full D-Pad controller and Keyboard navigation support (`Arrow keys`, `Enter`, `Spacebar`, `W/A/S/D`).
